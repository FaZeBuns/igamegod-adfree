# iGameGod-AdFree

**For iGameGod version 0.6.7 only.**

Free jailbreak tweak by **Buns** that blocks interstitial ads in iGameGod without breaking memory search or Live Offset Patcher.

Not affiliated with iOSGods.

## Install

Download the `.deb` from [Releases](https://github.com/FaZeBuns/igamegod-adfree/releases) and open it in **Sileo** or **Zebra**. Force-quit iGameGod after install.
